Publications

Smart Traffic Light Controller using Deep Reinforcement Learning

IEEE 3rd International Conference for Emerging Technologies

A. Abhishek, P. Nayak, K. P. Hegde, A. Lakshmi Prasad and K. S. Nagegowda, ”Smart Traffic Light Controller using Deep Reinforcement Learning,” 2022 3rd International Conference for Emerging Technology (INCET), Belgaum, India, 2022, pp. 1-5, doi: 10.1109/INCET54531.2022.9824501.

  • Proposed a deep reinforcement learning-based approach for adaptive traffic signal control, addressing real-world congestion challenges using simulation-driven experimentation.
  • The simulation environment is built using SUMO and interfaced through the TraCI API for real-time interaction.
  • Conducted extensive performance analysis of ANN and CNN models across varying intersection complexities, highlighting the impact of architecture and training depth.
  • State representation is handled using Discrete Traffic State Encoding (DTSE), with an epsilon-greedy policy and experience replay for training.
  • The model behavior is evaluated using metrics like queue length and cumulative delay, demonstrating measurable improvements over traditional signal timing strategies.
